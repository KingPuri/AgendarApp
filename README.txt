aplicacion para agendar o reservar una hora al doc

se utiliza React + Flask + MongoDB para el desarrollo de la app

Se puede registrar a través de un formulario,
además de poder ver una lista de los elementos indexados
además de poder elimarlos de ser necesario

enviar correo al indexar formulario

instrucciones
es necesario instalar mongoDB, python 3.10 (configuracion de path sin limites)
abrir backend en editor de texto(de preferencia VisualCodeStudio)
Abrir terminal, posicionar en la carpeta backend/src y poner siguiente comando
python app.py
Luego en otra terminar 
posicionar en fronted y escribir npm i
luego npm start
y ya deberia correr el programa sin problemas,
